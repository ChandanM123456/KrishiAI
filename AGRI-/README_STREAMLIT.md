# 🌾 KrishiAI Pro - Streamlit Community Cloud Deployment

## 🚀 Quick Deploy to Streamlit Community Cloud

### 📋 Prerequisites
- GitHub repository with the latest code
- Streamlit account
- All dependencies in `requirements.txt`

### 🛠️ Deployment Steps

#### Method 1: Direct GitHub Integration (Easiest)
1. **Go to Streamlit Community Cloud**: https://share.streamlit.io/
2. **Click "New app"** or "Deploy now"
3. **Connect GitHub**: Select your repository `ChandanM123456/KrishiAI`
4. **Select Branch**: Choose `main` branch
5. **Main File**: Set `streamlit_app.py` as main file
6. **Configure**: 
   - App name: `KrishiAI Pro`
   - App URL: `krishi-ai-pro`
7. **Deploy**: Click "Deploy" button

#### Method 2: Using Streamlit CLI
```bash
# Install Streamlit CLI
pip install streamlit

# Login to Streamlit
streamlit login

# Deploy from your repository
cd AGRI-
streamlit deploy
```
